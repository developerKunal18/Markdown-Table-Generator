# Markdown Table Generator

A Python tool that converts comma-separated input into a Markdown formatted table.

## Description
This script allows users to enter rows of data and automatically generates a Markdown table format that can be used in README files or documentation.

## Features
- Generates Markdown tables
- Simple input format
- Useful for GitHub documentation
- Beginner-friendly

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python table_generator.py

4. Enter rows separated by commas
5. Type `done` when finished

## Example

Input:
Name,Age,City  
John,25,New York  
Alice,30,London  

Output:

| Name | Age | City |
|------|-----|------|
| John | 25 | New York |
| Alice | 30 | London |
